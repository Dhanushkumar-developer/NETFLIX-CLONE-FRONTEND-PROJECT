🎬 Netflix Clone Website

A responsive Netflix Clone website built using HTML, CSS, JavaScript, and Bootstrap. The project recreates the Netflix user interface with multiple pages including login, signup, movies, anime, and subscription upgrade sections.

🚀 Features

- Modern Netflix-inspired UI
- Fully Responsive Design
- Home Page
- Login Page
- Sign Up Page
- Movies Section
- Anime Section
- Subscription Upgrade Page
- Mobile-Friendly Layout
- Smooth Navigation

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
